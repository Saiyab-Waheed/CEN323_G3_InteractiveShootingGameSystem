# Shooting Game with Score Management System

## Project Description
This project is a Shooting Game developed in 8086 Assembly Language using EMU8086. The game allows players to shoot targets by matching the target number. It includes score management, leaderboard sorting, grading, animations, streak bonus, and sound effects.

## Features
- Random target generation
- Hit/Miss detection
- Score tracking
- Leaderboard system
- Bubble sort ranking
- Grade calculation
- ASCII animations
- Sound effects using beep
- Streak bonus system

## Concepts Used
- Arrays
- Loops
- Procedures
- Macros
- Stack
- Interrupts
- Bubble Sort
- Indexed Addressing
- Conditional Jumps

## Tools Used
- EMU8086
- 8086 Assembly Language

## How to Run
1. Open EMU8086
2. Open shooting_game.asm
3. Click Assemble
4. Click Run
